## Decision

The estate adopts the compound-agent lifecycle contract authored at the
first instrumented compound run — the host estate's tracked research
document
`compound-agent-composition-invocation-and-information-preserving-dissolution-2026-07-11.md`
(Core-resident beside this record —
[the annex](../compound-agent-composition-invocation-and-information-preserving-dissolution-2026-07-11.md)
travels with the Core; relocated from the hydrating repo's research tier
2026-07-11 for practice-core portability, content conserved verbatim) —
as binding Practice doctrine. That tracked
document is this decision's normative annex: its §Terms (element / autonomous / invoked / directing /
custodian / side-effect executor), §Composition contract (the tracked
composition ledger with full canonical identities and per-element
attention contracts), §Operating contract (watcher-capable custody owns
ambient awareness; exactly one side-effect executor per window; invoked
elements never silently inherit authority), §Information-preserving
dissolution protocol (evidence freeze → pre-positioned obligations →
attended acceptance by named individual owners → authority-end instant →
independently reconstructable historical fabric), and §Reconstitution
(a new lineage edge, never edited history) govern every future compound.

Ratification reconciles rather than edits the related decisions above:
where the annex touches their contracts it composes with them, and any
future conflict resolves by amending the owning PDR, never by forking the
contract here.

## Warrant and evidence

The first run's permanent record
(`compound-agent-first-run-security-and-learning-2026-07-11.md`, the
first-read root) proved both halves: heterogeneous elements produced
genuinely stronger adversarial synthesis under an active directing/custody
topology, AND every operational failure (deaf watchers, HOLD-as-barrier,
custody without sole-writer execution, anti-convergent retirement) traced
to a contract this annex now makes explicit. The next compound run is
gated by the report's §Next-run acceptance gate under THIS contract.

## Falsifiability

- If a future compound run under this contract loses an obligation, an
  identity lineage, or evidence that the dissolution protocol claims to
  conserve, the contract is defective and this PDR is amended or
  superseded — never quietly waived.
- If the composition ledger or attended-acceptance steps prove to be
  ceremony (cost with no caught defect across two or more runs), the
  contract slims by amendment.
- The deferral alternative (ratify only after a second gated run) was
  put to the owner and declined 2026-07-11; a second run that
  contradicts the contract reopens it via amendment with that run's
  evidence.
