# Reading Discipline

All file paths in sub-agent templates are relative to the repository root.

## Universal Reading Requirements

Every installed agent MUST read and internalise these documents before doing review or analysis work:

| Document                          | Purpose                                                            |
| --------------------------------- | ------------------------------------------------------------------ |
| `.agent/directives/AGENT.md`      | Repo doctrine, current sub-agent roster, and quality-gate commands |
| `.agent/directives/principles.md` | Authoritative engineering doctrine                                 |

Templates may add stricter role-specific reading requirements on top of this baseline.

## The Discipline

Reading requirements are not optional.

If doctrine conflicts with the current code, report the conflict explicitly. Do not silently follow the code and do not silently ignore the doctrine.
