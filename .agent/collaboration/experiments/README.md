# Collaboration Experiments (tracked durable home)

Dated records of deliberate multi-agent collaboration experiments — protocol
trials, coordination-shape probes, and their observed outcomes — that deserve
a tracked home rather than the gitignored state plane. One dated markdown
file per experiment: `YYYY-MM-DD-<topic-slug>.md`.

Records here are evidence, not doctrine: what an experiment shows graduates
through the normal learning loop (napkin → distilled → rule/PDR), and the
experiment record stays as the citable worked instance. Live rapid-dialogue
channels are NOT experiments — they live in
[`../rapid-comms/`](../rapid-comms/README.md).

(Convention brought via Resonance from Oak, where `.agent/collaboration/experiments/` holds
the founding instances; each estate's dated records stay in that estate.)
