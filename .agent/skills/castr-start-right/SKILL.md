# Castr Start Right

## Goal

Re-anchor a Castr session on the repo's doctrine, current workstream, and quality gates before substantial work begins.

## Workflow

1. Read and follow `.agent/prompts/start-right.prompt.md`.
2. Read the directives referenced by that prompt.
3. Confirm the real current entrypoint from `session-entry.prompt.md`, `roadmap.md`, and the active or paused plan set.
4. Re-state the user impact, architectural layer, assumptions, and proof obligations before making changes.
