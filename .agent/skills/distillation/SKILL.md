# Distillation

## Goal

Extract high-signal patterns from the napkin into a shorter, more durable memory document.

## Workflow

1. Read the current napkin and distilled learnings.
2. Extract specific, actionable, non-obvious lessons.
3. Merge them into `.agent/memory/distilled.md`, updating existing entries rather than duplicating them.
4. Remove or trim entries that have already graduated into permanent docs.
5. Archive or reset the napkin only after the distilled version is in place.
